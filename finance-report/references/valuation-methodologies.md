# 估值方法论

多方法估值框架，确保每个标的有可靠的估值锚点。

## 原则

1. **至少用 2 种方法**（DCF + Comps 是最低要求）
2. **给区间，不给点估值**（Bear/Base/Bull）
3. **交叉验证**：DCF 隐含倍数 vs Comps 中位数
4. **所有假设必须有理由**，不是拍脑袋

## 方法一：DCF 分析

### 使用场景
- 所有核心持仓和高信心候选

### DCF 结果解读框架

```
              Bear       Base       Bull

```

### 合理性检查清单

| 检查项 | 标准 | 异常处理 |
|--------|------|---------|
| TV 占比 | 50-70% | >75% → 投射期可能太短或增速假设过低 |
| 隐含 EV/EBITDA | 与 Comps P25-P75 范围一致 | 超出 → 需解释溢价/折价理由 |
| WACC | 8-14% (成长股 10-14%) | 超出 → 检查 Beta 和资本结构 |
| 终端增速 | 2-3% (不超过 GDP) | >4% → 过于激进 |

## 方法二：可比公司分析

### 使用场景
- 所有扫描标的（有对比意义时）

### Comps 表解读框架

```
              EV/Rev  EV/EBITDA  Fwd PE  PEG   FCF Yield
[标的]        ___x    ___x       ___x    ___x  ___%
同业 Median   ___x    ___x       ___x    ___x  ___%
同业 P25      ___x    ___x       ___x    ___x  ___%
同业 P75      ___x    ___x       ___x    ___x  ___%
```

### 溢价/折价判断

| 情况 | 解读 | 前提条件 |
|------|------|---------|
| 低于 P25 | 可能被低估 | 增速/利润率不差于同业 |
| P25-Median | 合理偏低 | 一般情况 |
| Median-P75 | 合理偏高 | 增速/护城河优于同业 |
| 高于 P75 | 可能高估 | 需要极强的差异化理由 |

### 增速调整倍数

```
PEG = Forward PE / 预期增速%
PEG < 1.0  → 增速没被充分定价（潜在低估）
PEG 1.0-2.0 → 合理范围
PEG > 2.0  → 增速溢价过高
```

### 加权估值

| 方法 | 权重 | 估值 | 加权贡献 |
|------|------|------|---------|
| DCF Base | 50% | $___ | $___ |
| Comps Median | 35% | $___ | $___ |
| 分析师目标均价 | 15% | $___ | $___ |
| **加权目标价** | 100% | | **$___** |

**权重调整原则**：
- DCF 投射可靠时 → DCF 权重提升
- 同业可比性强 → Comps 权重提升
- 分析师覆盖多且分歧小 → 分析师权重可到 20%
- 分析师覆盖少或分歧大 → 降低或不用

## 估值输出要求

每个核心标的的估值部分必须包含：
1. DCF Bear/Base/Bull + 合理性检查
2. Comps 对比位置（vs P25/Median/P75）
3. PEG 或增速调整倍数
4. 足球场图
5. 加权目标价 + 隐含涨幅
6. 最差情景是否仍高于止损价
