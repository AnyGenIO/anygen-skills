# 竞争分析框架

用于赛道扫描和标的对比。

## 第一步：识别行业核心指标

不同赛道用不同指标。根据目标行业选 5-7 个最能区分赢家和输家的指标。

常见维度：

| 指标 | 为什么重要 |
|------|-----------|
| Revenue Growth YoY% | 增速是成长股最核心的指标 |
| Gross Margin% | 定价权和竞争壁垒的体现 |
| ROIC% | 资本回报效率，区分好公司和平庸公司 |
| Customer Concentration | 客户依赖度 = 风险 |
| R&D as % of Revenue | 研发强度 = 未来竞争力 |
| FCF Yield% | 自由现金流生成能力 |

注：AI 根据具体行业特点动态选择最相关的指标，不使用固定指标集。

## 第二步：竞争格局地图

### 定位矩阵

按两个最重要维度（通常是增速 vs 估值）定位：

```
              低估值 ←————————————→ 高估值
高增速 │  ★ 最佳机会        │ 高增长但贵    │
       │  (增速>同业, PE<同业)│ (需验证持续性) │
       │                    │              │
低增速 │  价值陷阱?          │ 避开          │
       │  (便宜有原因?)      │ (高估+慢增长)  │
```

### 赛道层级

根据研究结果动态分层：
- Tier 1 (平台级): 行业垄断者，生态锁定
- Tier 2 (核心供应): 高护城河的核心环节
- Tier 3 (高增长): 爆发增长期的新进者
- Tier 4 (周期性): 龙头但受周期影响
- Tier 5 (挑战者): 市场份额争夺中

## 第四步：市场份额与趋势

搜索获取市场份额数据。

关键问题：
- 谁在赢得份额？（扩大 = 好）
- 谁在失去份额？（收缩 = 危险信号）
- 市场是整合还是碎片化？
